# Codex Island v1.0.5

这是一次额度显示与胶囊交互稳定性更新。Windows、macOS 与 Linux 继续使用同一套 Rust 原生后台同步机制，并统一修复平台构建配置。

![Codex Island 展开额度面板](https://raw.githubusercontent.com/s840207702/codex-island/v1.0.5/docs/screenshots/codex-island-dashboard.jpg)

## 本次更新

- 修复胶囊与详情面板可能短暂显示不同额度的问题，继续通过原生后台任务按 60 秒周期同步最新周额度。
- 修复快速移出再移入胶囊时，旧定时器仍可能触发并让详情面板错误收起的问题。
- 禁止双击胶囊时选中文本，保持纯展示组件的交互体验。
- 将 macOS 私有 API feature 限定到 macOS 目标，恢复 Windows 构建与测试兼容性。
- 保留网络失败指数退避、恢复后自动刷新以及手动强制刷新机制。

## 使用条件

- Windows 10/11、macOS 11 或更高版本，或主流 x64 Linux 发行版。
- 本机已经登录 Codex，并在用户主目录中存在 `.codex/auth.json`。

> macOS 安装包使用 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次运行如果被 Gatekeeper 拦截，请在“系统设置 → 隐私与安全性”中手动允许。
