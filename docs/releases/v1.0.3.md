# Codex Island v1.0.3

这是 Codex Island 首个同时提供 Windows、macOS 与 Linux 安装包的公开版本。macOS 包为 Universal 2，同时兼容 Apple Silicon 与 Intel Mac。

![Codex Island 展开额度面板](https://raw.githubusercontent.com/s840207702/codex-island/v1.0.3/docs/screenshots/codex-island-dashboard.jpg)

## 本次更新

- 新增 Windows x64、macOS Universal 2 与 Linux x64 安装包。
- 修复透明窗口周围的白色矩形、浅色描边和字体显示问题。
- 常驻胶囊自动避开 MacBook 物理刘海与菜单栏安全区域。
- 详情面板支持悬停展开、离开收起，并修复多实例导致的无法折叠问题。
- macOS 沉浸模式改为按前台窗口是否完整覆盖胶囊触发，普通最大化窗口也能进入沉浸状态。
- macOS 与 Windows 保持共享 UI 和额度状态逻辑，平台原生窗口行为分别处理。

## 使用条件

- Windows 10/11、macOS 11 或更高版本，或主流 x64 Linux 发行版。
- 本机已经登录 Codex，并存在 `~/.codex/auth.json`。

> 当前安装包使用 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次运行如果被 Gatekeeper 拦截，请在“系统设置 → 隐私与安全性”中手动允许。
