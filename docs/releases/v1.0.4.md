# Codex Island v1.0.4

这是一次跨平台额度刷新可靠性更新，Windows、macOS 与 Linux 使用同一套 Rust 原生后台同步机制。

![Codex Island 展开额度面板](https://raw.githubusercontent.com/s840207702/codex-island/v1.0.4/docs/screenshots/codex-island-dashboard.jpg)

## 本次更新

- 将自动刷新从 WebView 计时器迁移到 Rust 原生后台任务，避免窗口被遮挡或长期不活跃时停止同步。
- 正常状态下每 60 秒同步额度；失败后持续指数退避，网络恢复后可自动回到正常刷新。
- macOS 睡眠唤醒后按系统时间及时补刷新，不再长期停留在旧额度快照。
- 修复详情面板“立即刷新”只读取内存缓存、没有真正请求服务端的问题。
- Windows 与 macOS 共用相同的刷新状态机，保留各自的平台窗口行为。

## 使用条件

- Windows 10/11、macOS 11 或更高版本，或主流 x64 Linux 发行版。
- 本机已经登录 Codex，并在用户主目录中存在 `.codex/auth.json`。

> macOS 安装包使用 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次运行如果被 Gatekeeper 拦截，请在“系统设置 → 隐私与安全性”中手动允许。
